OntoTrek is an OBOFoundry.org ontology terminology viewer that takes advantage of 3d graph rendering software. We have a few ontologies to choose from during this alpha/beta phase - some easy to load like IDO, some too big, like CLO and GenEpiO - they don't fully organize before the force placement algorithm ends. Some ontologies demonstrate nice organization under the Basic Formal Ontology (BFO, whose 34 nodes are big, sunny, and always in the same position), and others show an "independent streak". OntoTrek color codes terms by their ontology prefix so that one can visually see which ontologies are at play. Currently it displays all class-subclass relations starting from owl:Entity or BFO:0000001 Entity. (Its actually running off of flat JSON versions of the given ontologies, which were generated using the ontofetch.py program).

Changes in the presentation of the graph are planned so that a more hierarchic, mountain-like display with upper level terms is shown, enabling users to have a stable (BFO) reference point over time.

Use mouse or trackpad to rotate, zoom, or pan the display. Click on a node to focus on it; rotation will occur around that item. Clicking on a link will take you to its child node. Click on dashboard parent to navigate up to top.

This software is being developed at the Hsiao Lab which is associated with the University of British Columbia Department of Pathology and Laboratory Medicine, and with the PHSA Public Health Laboratory at the British Columbia Centre For Disease Control. The project's GitHub is here. Contact Damion Dooley for more information.
